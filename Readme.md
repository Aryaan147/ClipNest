## What is ClipNest?

ClipNest is a smart clipboard manager built for people who copy the same things over and over again вҖ” and are tired of digging through notes, browser history, or Notion pages to find them.

Whether it's a YouTube channel you share with everyone, a research paper link, your social handles, a repo URL, or a code snippet you paste daily вҖ” **ClipNest stores it once, copies it in one tap.**

No friction. No searching. Just tap and move on.

---

## вңЁ Features

- **One-tap copy** вҖ” Every item copies to clipboard instantly, no extra steps
- **Search** вҖ” Find any clip across all categories instantly
- **Categories** вҖ” Group your clips however you think: by project, type, context, or anything else
- **Universal content** вҖ” Links, code snippets, prompts, social IDs, research papers, repo URLs, anything text-based
- **Offline support** вҖ” Your clips are available even without internet; syncs automatically when back online
- **Sleek UI** вҖ” Clean, focused interface designed to stay out of your way
- **Authentication** вҖ” Your clips are yours; secure login keeps data private
- **Cloud sync via Firebase** вҖ” Access your clips from any device, always up to date
- **Cross-platform** вҖ” iOS, Android, and Web from a single codebase (Expo)

## рҹҡҖ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) (v18 or higher)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)
- A Firebase project with Firestore and Authentication enabled

### Installation

```bash
# Clone the repository
git clone https://github.com/Aryaan147/ClipNest.git
cd ClipNest

# Install dependencies
npm install

# Start the development server
npx expo start
```

### Firebase Setup

1. Create a project at [Firebase Console](https://console.firebase.google.com)
2. Enable **Firestore Database** and **Authentication** (Email/Password or your preferred provider)
3. In Firestore settings, enable **offline persistence** for offline support
4. Copy your Firebase config and create a `.env` file in the root:

```env
EXPO_PUBLIC_FIREBASE_API_KEY=your_api_key
EXPO_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
EXPO_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
EXPO_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
EXPO_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
EXPO_PUBLIC_FIREBASE_APP_ID=your_app_id
```

5. Run the app:

```bash
# Start development server
npx expo start

# iOS
npx expo run:ios

# Android
npx expo run:android

# Web
npx expo start --web
```
